# Kirov Monitoring SDK

Shared monitoring & observability SDK for the Kirov Security ecosystem. Provides structured logging, metrics collection, and telemetry primitives for Python services.

## Installation

```bash
pip install kirov-monitoring-sdk
```

## Features

- Structured JSON logging via structlog
- Pydantic-based metric models
- FastAPI middleware for request tracing
- OpenTelemetry-compatible spans
